Purpose

This site showcases the rich diversity of African musical instruments. From drums to stringed instruments, each piece carries cultural history and deep artistic expression. It serves to inform, celebrate, and preserve African heritage.

Creator

Stephen Amankwaa Atta is a passionate software development student at BYU-Idaho with a love for music and cultural storytelling. This website is a WDD 131 project to apply HTML, CSS, and responsive design skills.

Mission

Our mission is to educate and inspire by showcasing Africa’s musical heritage. We aim to foster appreciation for traditional instruments and ensure their legacy is preserved and shared across cultures.

Technologies Used

  • HTML5 & CSS3 for structure and styling
  • Responsive Design principles using media queries
  • JavaScript for interactive elements like date updates
  • Google Fonts for modern typography
